## Step 1 — Push to GitHub (2 minutes)

### Option A: GitHub Desktop (easiest)
1. Open GitHub Desktop
2. File → Add Local Repository → `C:\Users\timot\.gemini\antigravity\scratch\nexus-veil`
3. Click **Publish repository**
4. Name it: `sml-argus`
5. Set to **Private** (recommended)
6. Done

### Option B: Command line (if you have a GitHub token)
```bash
# Create repo on GitHub first, then:
git remote add origin https://github.com/YOUR_USERNAME/sml-argus.git
git branch -M main
git push -u origin main
```

## Step 2 — Deploy to Render (3 minutes)

1. Go to **https://render.com**
2. Click **New → Blueprint**
3. Connect your GitHub account
4. Select the `sml-argus` repo
5. Render reads `render.yaml` automatically — click **Apply**
6. It creates:
   - Web service: `sml-argus`
   - Postgres database: `argus-db` (free tier)

### Set environment variables in Render dashboard:
| Variable | Value |
|---|---|
| `DISCORD_WEBHOOK_URL` | Your Discord webhook URL |
| `SECRET_KEY` | (auto-generated by Render) |
| `DATABASE_URL` | (auto-set from your Postgres DB) |

---

## Step 3 — Set your Discord Webhook (2 minutes)

1. In your Discord server → Edit Channel → Integrations → Webhooks
2. Create webhook → Copy URL
3. Paste into Render environment variable: `DISCORD_WEBHOOK_URL`
